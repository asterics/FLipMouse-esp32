Detailed Description

FUNCTION - Infrared remotes.

This module is used for recording & sending of infrared commands. Pinning and low-level interfacing for infrared is done in hal_io.c, this part here manages loading & storing the commands.

FUNCTION - Trigger an IR command recording.

This method is used to record one infrared command. It will block until either the timeout is reached or a command is received. The command is stored vial hal_storage.
