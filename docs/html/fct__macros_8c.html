Detailed Description

FUNCTION - Execute macros.

This module is used for sending macros to the command parser.

Macros in context of FLipMouse/FABI are a string of concatenated AT commands, equal to normal AT commands sent by the host.

Note
AT command separation is done by a semicolon (';')
Warning
If you want to write the semicolon within a macro, use AT KP with the corresponding keycode!
