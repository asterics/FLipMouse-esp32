Detailed Description

Event Handler - HID.

This module is used as an event handler for all HID based virtual buttons, containing:

  • Mouse clicking, holding, pressing & release as well as movement (X/Y/Z)
  • Keyboard press,hold,release & write
  • Joystick press,hold,release & axis movement

handler_hid_init is initializing the mutex for adding a command to the chained list and adding handler_hid to the system event queue.

Note
Currently, we use the system event queue (because there is already a task attached). Maybe we switch to an unique one.
Mouse/keyboard/joystick control by mouthpiece is done in hal_adc!
See also
hal_adc

esp_err_t handler_hid_addCmd ( hid_cmd_t *  newCmd,
uint8_t  replace 
)

Add a new HID command for a virtual button.

This method adds the given HID command to the list of HID commands which will be processed if the corresponding VB is triggered.

Note
Highest bit determines press/release action. If set, it is a press!
If VB number is set to VB_SINGLESHOT, the command will be sent immediately.
We will malloc for each command here. To free the memory, call handler_hid_clearCmds .
Parameters
newCmdNew command to be added.
replaceIf set to != 0, any previously assigned command is removed from list.
Returns
ESP_OK if added, ESP_FAIL if not added (out of memory)
