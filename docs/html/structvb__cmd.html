Detailed Description

One VB command (not HID), maybe an element of a command chain.

This struct is used for VB commands, which are not HID commands (e.g. calibrate, IR recv/send)

uint8_t vb_cmd::vb

Number of virtual button. MSB signals if this is a press or release action:

  • If it is set (mask: 0x80), it is a press action
  • If it is cleared, it is a release action
