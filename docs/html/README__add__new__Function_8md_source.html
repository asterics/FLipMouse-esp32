1 # Howto add a new function
2 
3 If you want to add a new function to the FLipMouse/FABI, you have to take care of following steps:
4 
5 ## Implement your function in handler_hid/handler_vb
6 
7 Please use the corresponding handler. handler_hid for anything related to HID, which will be sent via I2C to the LPC USB chip.
8 handler_vb should be responsible for any other actions. If you want to implement a handler on your own, please take care of
9 adding your handler to the command parser.
10 
11 If you want to add your function to the existing handler, add your type of function to vb_cmd_type_t (handler_vb) and in the
12 handler_vb function.
13 
14 ## Add command to command parser
15 
16 The serial command parsing from the host's commands (based on an AT command set) is done in task_commands.c.
17 If you want to add a new function, add it to the onecmd_t commands[] struct.
18 
19 In this struct, you can define a field in the generalConfig_t array, which will be modified by the command parser.
20 You don't have to declare anything further.
21 
22 The second possibility is a command handler function, which you can declare in this struct. If your
23 command is recognized and the parameter check passes, your handler will be called.
